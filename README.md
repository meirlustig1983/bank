# Bank Management

Bank management is a some type of sand-box repository to try a new technologies

## List of technologies

* Java JDK-17
* Gradle 7.6.1
* Spring Boot 3.0.6
* Spring Web Services
* Spring Data JPA
* Spring Web
* Rest Repositories
* Lombok 1.18.26
* Junit
* Git CI-CD
* PostgreSQL
